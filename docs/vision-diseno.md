# **App Name**: D&D Adventures AI

## Core Features:

- AI Dungeon Master: Gemini 2.5 AI acts as a Dungeon Master, interpreting player actions and narrating the game world based on D&D 5e rules. The AI acts as a tool for parsing user actions.
- Character Management: Creation of player characters, either manually or via AI generation, managing character stats, skills, and inventory. Accommodates up to 4 characters, controlled by either a human or AI.
- Monster Generator & API Integration: Generates monsters using internal algorithms and scales them dynamically based on the level of the adventuring party. Connects to a public D&D 5e API to obtain monster stat blocks and descriptions.
- Dungeon Mapping: AI tracks the game world including dungeons and overworld locations. Displays text descriptions of the party's current location, points of interest, and potential threats.
- Combat System: Implements turn-based combat, calculating hits, damage, and applying status conditions from the D&D 5e ruleset.
- Dice Rolling System: Virtual dice roller that automatically calculates results according to D&D 5e mechanics, including advantage, disadvantage, and critical hits. The AI Dungeon master can trigger rolls, as well as the player
- Adventure Parsing & Translation: Loads adventures from flexible JSON formats. Automatically parses JSON adventure data and translates adventures from English to Spanish.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) evoking a sense of mystery and adventure, reflecting the fantasy setting of D&D.
- Background color: Light grey (#E8EAF6), providing a clean and unobtrusive backdrop that ensures readability.
- Accent color: Amber (#FFB300), used sparingly to draw attention to important interactive elements and calls to action.
- Body and headline font: 'Literata', a transitional serif, bringing a literary and slightly formal feel appropriate for the setting of D&D.
- Icons: Use icons related to D&D 5e such as swords, shields, and maps, designed with a modern and clean aesthetic.
- The layout should be clean and intuitive, with clear sections for character information, game narrative, and dice rolling.
- Subtle animations for dice rolls, combat sequences, and when loading new adventure segments to enhance user engagement.
